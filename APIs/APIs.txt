Métodos HTTP
   GET
      a API fornece informações
   POST
      a API recebe informações que devem ou não ser registradas
   DELETE
      a API recebe indetificadores de registros que devem ser apagados
   PUT
      a API recebe informações para update de um ou  ais registros
   PATCH
      a API recebe informações para update de um registro
