Regras para escrever nome de tabelas e de campos:
   1- Deve começar por um letra do alfabeto
   2- Os caracteres seguintes não são permitidos:   () + - / * " ; = & | # < > ^ ' {} %
   3- Não pode conter espaços
   4- Não pode conter acentuação

Tipos de campos:
   Text
   Number
   Datatime
   Primary Key


SELECT

   Iguais(=)

   Maior que(>)
   Menor que(<)

   Menor ou Igual(<=)
   Não igual a(<>)
   Diferente de(!=)

   Adição(+)
