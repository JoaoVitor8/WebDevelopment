Básico:

git clone + Repositório (Vá até o GitHub, depois no repositório desejado, clique em "Code" e copie o link no SSH)- Clona o repositorio que deseja usar
git status - Vê se algo foi alterado
git add . - Adiciona tudo em que eu fiz alteração para o proximo commit
git commit -m "Comentario do Commit" - Estou commitando
git push - Vai sicronizar o commit com o repositório

git pull - Atualiza para a vesão mais recente (Caso tenha mais pessoas alem de vc mechendo no repositório)
